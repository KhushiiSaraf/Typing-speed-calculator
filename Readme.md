# TypeMaster ⌨️

A full stack typing speed web app with real time WPM, CPM and accuracy tracking.

🔗 **Live Demo:** [TypeMaster](https://typing-speed-calculator-liart.vercel.app/)

## Features
- **Normal Mode** — Type against a timer of your choice, tracks WPM, CPM and accuracy in real time
- **Survival Mode** — Survive as long as you can. Correct words add 2 seconds, wrong ones take away 5. No backspace allowed
- **AI Feedback** — Personalized feedback on your mistake patterns after each test
- **Authentication** — Login and signup to save your best scores
- **Leaderboard** — Compete with other users on Survival Mode

## Tech Stack
**Frontend:** React, Vite, Tailwind CSS  
**Backend:** Node.js, Express  
**Database:** MongoDB Atlas  
**Other:** JWT Authentication, OpenAI API  
**Deployed on:** Vercel and Render
